# Blockchain Project from scratch by Golang

A simple blockchain project from scratch by Golang

## Quick start

```bash
git clone https://github.com/hmthanh/blockchain && cd blockchain

make all
make run
```

## Build

```bash
make build
```

## Test

Using Makefile

```bash
make test
```

or

```bash
go test ./...
```

## Public

```bash
git tag v0.1.0
git push origin v0.1.0
```
